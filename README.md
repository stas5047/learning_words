# Word Learning Web Application

**Project Description**  
This is a web application for learning foreign words and phrases, designed for users who want to efficiently train their memory and expand their vocabulary. The app combines a convenient interface for managing a word list with interactive tests to check knowledge.

---

## **Features**

### 1. Home Page
- Navigation to the main sections of the app: Add Words, Testing, Word List.
- Option to switch the interface theme (e.g., light/dark mode).

### 2. Word Management
- **Add Word/Phrase:** Users can enter a foreign word and its translation.  
- **Import Word List:** Upload a text file with words and translations (e.g., `word - translation`).  
- **Edit Words:** Modify existing entries.  
- **Delete Words:** Remove words with confirmation.  
- **Search:** Quickly search by word or translation.  

### 3. Testing
- **Multiple Choice:** A word is displayed, and users choose the correct translation from 4 options.  
- **Text Input:** A translation is displayed, and the user types the corresponding word.  
- Score tracking and display of test progress.  
- Final message showing the number of correct answers.

### 4. History and Progress
- Tracks completed tests and results.  
- Users can retake tests to reinforce knowledge.

### 5. Interface Settings
- Switch between light and dark themes.  
- Responsive design for desktop and mobile devices.  
- Highlights correct and incorrect answers during tests.

### 6. UX and Convenience
- Interactive notifications: word successfully added, input errors, test completion.  
- Minimalist and intuitive interface.  
- Support for loading large text files with multiple words.

---

## **Application Advantages**
- Data is stored locally or on a server, making word list management easy.  
- Can be adapted for any language or word database.  
- Simple structure allows easy future expansion (e.g., categories, statistics, spaced repetition).

---

## **Logical Structure of the Application**
1. Home page with navigation to sections.
2. Add Word page with word input and file upload.
3. Word List page with editing and deleting functionality.
4. Testing page with multiple choice and text input modes.
5. Notifications and answer highlighting system.
6. Interface theme settings.

---

## **Use Cases**
- For students learning foreign languages.  
- For self-study and reinforcement of learned vocabulary.  
- For memory training and vocabulary retention.

---

## **Project Goal**
To create a web application that combines word list management and interactive testing, allowing users to easily add words, check their knowledge, and track progress, while keeping the interface user-friendly and visually clear.
